# SelfManager

SelfManager is a personal life management Android application.
It starts with personal finance tracking and will expand into
todo list, habit tracker, and notes.

## Features (v1)
- Income & expense tracking
- Transaction history
- Monthly report
- Local SQLite database

## Tech Stack
- Java
- Android Studio
- SQLite
- Material Design

## Roadmap
- v2: Todo List
- v3: Habit Tracker
- Cloud Backup (Google Drive)
